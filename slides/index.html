So what is RxJS and why do I care?
Personally, I find RxJS much easier to understand through examples...
Imagine we had an HTTP client that returns an Observable of a GET request... 

							const users$ = http.get('api/users');
We can subscribe to that observable and provide a callback handler to process the result: 

							users$.subscribe(users => updateUsers(users));
We can also transform the Observable through the `pipe` method: 

							const todd$ = http.get('api/users').pipe(
								map(users => users.find(x => x.name === 'Todd'))
							);
You can create some really awesome, complex data pipelines with RxJS.
Even though it handles really complicated scenarios, the way it works is actually quite simple!
